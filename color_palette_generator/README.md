# Color Palette Generator

A simple front-end web app that generates random 5-color palettes with one click.

Live Demo: [ Color Palette Generator](https://fantastic-bavarois-3e6cc6.netlify.app)

## Features

- Generate a random 5-color palette instantly
- Click the copy icon or the color swatch to copy a hex code to clipboard
- Visual checkmark feedback on successful copy

## Tech Stack

- HTML, CSS, JavaScript (vanilla)
- [Font Awesome 6](https://fontawesome.com/) for icons

## Usage

1. Click **Generate Palette** to get 5 random colors
2. Click any color swatch or the copy icon to copy its hex code to your clipboard
